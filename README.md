# Computer-Vision-For-Extracting-Construction-Graphs-From-PDFs
End-to-end computer vision pipeline for extracting construction graphics and text from architectural floor plan PDFs. Uses PyMuPDF (600 DPI rasterization), YOLOv11s-seg instance segmentation, and rotation-aware EasyOCR with pre/post-processing. Final Year Project, Multimedia University Melaka..
